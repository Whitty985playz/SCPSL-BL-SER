	**LOCAL HOST:** *`C:\Users\[USER]\AppData\Roaming\SCP Secret Laboratory\LabAPI\configs\Scripted Events Reloaded`*


- Create an **`audio`** directory in your LabAPI folder.

	**DEDICATED SERVER:** *`~/.config/SCP Secret Laboratory/LabAPI/`*

	**LOCAL HOST:** *`C:\Users\[USER]\AppData\Roaming\SCP Secret Laboratory\LabAPI\`*

- Take the audio file in **_Audio** and put it in the newly created **Audio Directory**.

	**DEDICATED SERVER:** *`~/.config/SCP Secret Laboratory/LabAPI/audio`*

	**LOCAL HOST:** *`C:\Users\[USER]\AppData\Roaming\SCP Secret Laboratory\LabAPI\audio`*


- Soft restart your server(if online) and you should see the command "blacklight" if done correctly.
